# ⚡ GridBrain AI

<div align="center">

### AI-Powered EV Grid Intelligence & Smart Charging Management Platform

GridBrain AI is a next-generation EV charging intelligence platform designed for utility operators, planners, regulators, and smart city ecosystems.

The platform combines real-time grid monitoring, AI-driven analytics, explainable AI, infrastructure planning, and simulation intelligence into one unified dashboard.

</div>

---

# 🚀 Overview

As EV adoption rapidly increases, power grids face major challenges related to overload risks, peak demand, infrastructure planning, and operational visibility.

GridBrain AI solves these challenges through:

* ⚡ Real-time EV grid monitoring
* 🤖 AI-powered risk prediction
* 📊 Demand forecasting & analytics
* 🧠 Explainable AI (XAI)
* 🗺️ Smart charger planning & heatmaps
* 🧪 Scenario simulation engine
* 🏛️ Government & compliance dashboards
* 🌐 English + Kannada localization

# ⚙️ Installation & Setup

## Clone Repository

```bash
git clone https://github.com/your-username/GridBrain-AI.git
cd GridBrain-AI
```

## Install Dependencies

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

## Build for Production

```bash
npm run build
```
